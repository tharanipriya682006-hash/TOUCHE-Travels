# TOUCHÉ Travels

A responsive travel website developed using HTML5, CSS3, and JavaScript. The website allows users to explore popular destinations, browse travel packages, view a photo gallery, and book trips through a modern and user-friendly interface.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Home, About, Destinations, Packages, Gallery, Booking, and Contact pages
- Interactive booking form with client-side validation
- Responsive navigation menu
- Scroll-to-top button
- Accessible design with descriptive alt text and ARIA labels
- Clean and modern travel-themed interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
